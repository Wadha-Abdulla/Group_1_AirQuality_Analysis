# California Air Quality Analysis (Group 1)

##  Project Description
This project analyzes California air quality data from 2020 using Python in Google Colab.  
The dataset includes PM2.5 concentration levels, AQI values, and location information across different regions in California.

The project focuses on:
- Exploring air quality patterns
- Filtering and sorting data
- Grouping and analyzing regions
- Classifying air quality into Good, Moderate, and Unhealthy categories
